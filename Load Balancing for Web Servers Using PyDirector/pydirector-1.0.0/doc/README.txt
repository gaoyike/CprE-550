Documentation for Python director
---------------------------------

The docs directory contains the following files:

configure.txt

  - Basic configuration guide to the python director

schedulers.txt

  - Information about the different schedulers available in python
    director.

xmlreference.txt

  - a reference guide for the XML config file format.

webapi.txt

  - documentation of all web API calls.

pythondirector.dtd

  - XML DTD for the python director config file XML.

balance.fig

  - xfig image of "what is a load balancer"

LICENSE.txt

  - license for python director.

ChangeLog

  - current GNU-style Changelog.

OldChangelog.txt

  - Changelog prior to first SF import.
